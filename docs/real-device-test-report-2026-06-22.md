## 发现的问题

### P1 — 必须修复

1. **消息删除误触**
   - 问题：长按消息触发删除，而非预期功能
   - 建议：调整长按触发区域或增加防误触机制

### P2 — 建议修复

1. **输入法兼容性**
   - 问题：ADB input text 无法输入中文
   - 说明：这是 ADB 限制，非应用问题
   - 建议：手动测试验证中文输入

2. **设置页面导出**
   - 问题：SettingsActivity 未设置 `exported=true`
   - 建议：在 AndroidManifest.xml 中配置
