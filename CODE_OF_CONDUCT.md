# 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：参与本项目的每个人，无论年龄、体型、残疾、族裔、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教信仰或性取向如何，都不会受到骚扰。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性暗示的语言或图像，以及不受欢迎的性关注
- 恶意评论、人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中被合理认为不当的行为

## 适用范围

本行为准则适用于本项目的所有场景，包括：

- 项目仓库（Issues、Pull Requests、Discussions）
- 社交媒体上的项目相关讨论
- 公共场合中代表项目的发言

## 执行

如果发现违反行为准则的行为，请通过以下方式报告：

- 发送邮件至：[your-email@example.com]
- 或在 Issue 中私下联系维护者

所有报告都将被保密处理。维护者有权移除任何违反行为准则的内容和用户。

## 执行指南

维护者将遵循以下指南来执行行为准则：

1. **纠正**：使用友好的语言指出不适当的行为
2. **警告**：对首次违规进行警告
3. **临时封禁**：对重复违规进行临时封禁
4. **永久封禁**：对严重违规或屡次违规进行永久封禁
