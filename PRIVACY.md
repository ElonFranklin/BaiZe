# 隐私政策

**最后更新：** 2026-06-22

## 概述

白泽（BaiZe）是一个本地运行的 AI 伙伴应用，我们非常重视您的隐私保护。本隐私政策说明了我们如何收集、使用和保护您的信息。

## 数据收集

### 我们收集的数据

**白泽不主动收集任何用户数据。**

所有数据均存储在您的设备本地，不会上传到任何服务器。

### 您主动提供的数据

- **API 配置**：如果您选择使用云端 API，您需要提供自己的 API Key。这些信息仅存储在您的设备上，不会上传。
- **聊天内容**：您与 AI 伙伴的对话内容仅存储在本地数据库中。
- **灵魂文件**：您导入或创建的灵魂文件仅存储在本地。

## 数据存储

### 存储位置

所有数据均存储在您的 Android 设备本地：

- **聊天记录**：SQLite 数据库（`app_databases/` 目录）
- **灵魂文件**：内部存储（`files/souls/` 目录）
- **设置信息**：SharedPreferences

### 数据安全

- **加密存储**：敏感数据（如 API Key）使用 AES-256-GCM 加密
- **本地存储**：所有数据不离开您的设备
- **备份保护**：`android:allowBackup="false"` 防止数据通过 ADB 备份泄露

## 数据使用

### 我们如何使用您的数据

**我们不使用您的任何数据。**

所有数据仅在您的设备上用于：

- 提供 AI 对话功能
- 维护您的灵魂文件和记忆
- 保存您的应用设置

### 数据共享

**我们不与任何第三方共享您的数据。**

- 您的聊天记录不会被上传
- 您的 API Key 不会被传输（除非您主动配置云端 API）
- 您的灵魂文件不会被分享

## 第三方服务

### 云端 API（可选）

如果您选择使用云端 API（如 OpenAI、Anthropic），请注意：

- 您需要自行提供 API Key
- API 请求会发送到相应的服务提供商
- 这些请求受第三方服务的隐私政策约束
- 我们不记录或存储这些请求的内容

### 模型下载

如果您选择下载本地模型：

- 模型文件从开源平台（如 HuggingFace）下载
- 下载过程不包含您的个人信息
- 模型文件存储在本地，不会被上传

## 权限说明

### 应用权限

| 权限 | 用途 | 是否必要 |
|------|------|----------|
| `INTERNET` | 云端 API 调用、模型下载 | 可选（仅云端功能需要）|
| `ACCESS_NETWORK_STATE` | 检测网络状态 | 是 |
| `RECORD_AUDIO` | 语音输入功能 | 可选（仅语音输入需要）|
| `READ_EXTERNAL_STORAGE` | 读取旧版模型文件 | 否（仅 Android 12 及以下）|
| `WRITE_EXTERNAL_STORAGE` | 写入旧版模型文件 | 否（仅 Android 10 及以下）|

### 权限管理

- 运行时权限：`RECORD_AUDIO` 需要用户明确授权
- 权限限制：存储权限已通过 `maxSdkVersion` 限制
- 无多余权限：不申请 `CAMERA`、`LOCATION`、`CONTACTS` 等无关权限

## 数据删除

### 删除您的数据

您可以随时删除您的数据：

1. **删除聊天记录**：在应用设置中选择"清除聊天记录"
2. **删除灵魂文件**：在灵魂管理界面删除
3. **卸载应用**：卸载应用将删除所有本地数据

### 数据保留

- 卸载应用后，所有数据将被删除
- 我们不保留任何用户数据副本

## 儿童隐私

白泽不针对 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。

## 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内发布，并更新"最后更新"日期。
